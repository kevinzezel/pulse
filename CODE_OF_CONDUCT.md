# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

The full text is available here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Short version

- Be respectful. Assume good intent. Disagree with ideas, not people.
- Keep discussion on-topic, constructive, and free of personal attacks.
- No spam, no off-topic self-promotion.

## Reporting

To report behavior that violates this Code, contact the maintainer:

- GitHub: <https://github.com/kevinzezel>
- Email: `kevin@adsscanner.com` (use subject `[pulse-conduct]`)

All reports are handled privately and in good faith.

## Scope

Applies to all project spaces (GitHub issues, PRs, Discussions, Discord if one is created, and any community channels explicitly associated with Pulse).
